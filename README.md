## 📁 Project Structure

- **CareNova/**
  - 🖥️ **frontend/** – React + Tailwind CSS + Framer Motion code  
    - 📂 **public/** – Static assets (images, icons, favicon)  
    - 📂 **src/** – React source code (components, pages, styles)  
  - ⚙️ **backend/** – Node.js + Express + PostgreSQL  
    - 📂 **controllers/** – API logic and request handling  
    - 📂 **models/** – Database schemas and models  
    - 📂 **routes/** – API route definitions  
    - **server.js** – Main backend server entry point  
  - 🔗 **smart-contracts/** – Blockchain verification (zkTLS / smart contracts)  
  - ⌚ **wearable-integration/** – Fitbit & Apple HealthKit SDK connectors  
  - 📄 **docs/** – Documentation, diagrams, screenshots, flowcharts  
  - 📜 **package.json** – Project dependencies & scripts  
  - 📝 **README.md** – Project overview and instructions  
  - 🚫 **.gitignore** – Files/folders to ignore in Git
